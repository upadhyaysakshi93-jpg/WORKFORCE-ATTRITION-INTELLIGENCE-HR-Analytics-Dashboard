# Dashboard Visual Evidence

This folder contains all the key visual snapshots from the Power BI dashboard used in this project. These visuals support the insight narratives described in the main README.

## List of Visuals Included

- `dashboard_home(page 1).png` – Executive Overview Dashboard
- `dashboard_home(page 2)` – Deep Dive in Dashboard
- `dashboard_kpis.png` – KPI Cards View
- `dashboard_tooltip.png` – Tooltip of Dashboard

> These visuals are intentionally kept separate from the main README to keep the project clean, structured, and easy to navigate.
